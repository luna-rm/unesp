
ARQUIVO AJUDA

O Cosmofy eh um programa de cadastro de musicas. 
O programa possui um menu com:

a) Um cadastro que possui 5 entradas, que sao

1.Nome da musica
2.Artista
3.Album
4.Duracao
5.Data de Lancamento

b) Opcao de alterar arquivo, que consiste em mudar alguma informacao de uma musica ja cadastrada

c) Opcao de apagar uma musica 

d) Opcao de consultar por uma musica especifica

e) Opcao de ver todas as musicas cadastradas

----------------------------------------------------------------------------

Tabelinha de comandos!! :)

Precione:

    - <seta para cima> para subir;
    - <seta para baixo> para descer;
    - <seta para esquerda> para voltar;
    - <seta para direita> para entrar na pasta;
    - "c" para criar uma nova pasta;
    - "r" para renomear uma pasta;
    - "d" para excluir uma pasta;
    - "h" para mudar o horario do sistema;
    - "esc" para sair;
    - "F1" para ajuda;
